# 📚 Документация CLI: Gemini и Codex

Этот документ содержит краткое руководство и статус проверки инструментов **Gemini CLI** и **Codex CLI**, используемых в этом фреймворке.

---

## ♊ Gemini CLI (`@google/gemini-cli`)

Интерактивный AI-агент от Google для разработки прямо в терминале.

### 🚀 Установка
Официальный пакет: `@google/gemini-cli`.
```bash
npm install -g @google/gemini-cli
```
*Статус на этой машине:* Работает через `npx`. Команда для вызова: `npx @google/gemini-cli`.

### 🛠 Основные команды
- `gemini`: Запуск интерактивной сессии.
- `gemini -p "Запрос"`: Выполнение разового промпта без входа в чат.
- `gemini --help`: Показать все опции (модели, контекст, инструменты).

### 📑 Особенности
- **Доступ к файлам:** Умеет читать, писать и проводить поиск по локальным файлам.
- **Инструменты Shell:** Выполняет команды терминала (тесты, сборка).
- **Поиск в сети:** Интегрированный Google Search и fetch содержимого страниц.
- **Поддержка MCP:** Работает с серверами Model Context Protocol.

---

## 🤖 Codex CLI (`@openai/codex`)

Легковесный агент для кодинга от OpenAI, работающий в цикле "Reason and Act".

### 🚀 Установка
Официальный пакет: `@openai/codex`.
```bash
npm install -g @openai/codex
```
*Статус на этой машине:* Работает через `npx`. Команда для вызова: `npx @openai/codex`.

### 🛠 Основные команды
- `codex`: Запуск интерактивного цикла агента.
- `codex --help`: Список конфигураций и параметров запуска.
- `codex -V`: Проверка версии.

### 📑 Особенности
- **Автономный кодинг:** Самостоятельно вносит правки и исправляет ошибки.
- **Оркестрация подзадач:** Поддерживает запуск специализированных подагентов.
- **Безопасность:** Режимы подтверждения опасных действий.
